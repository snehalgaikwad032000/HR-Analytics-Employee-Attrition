# HR Analytics & Employee Attrition Analysis

## Project Overview

This project analyses employee attrition trends and workforce behaviour using MySQL and Power BI. The objective was to identify key factors contributing to attrition and create an interactive dashboard for HR reporting and decision-making.

---

## Tools Used

* MySQL
* Power BI
* DAX
* SQL
* Data Visualization

---

## Key Business Insights

* Sales department showed the highest attrition rate.
* Employees working overtime had significantly higher attrition.
* Laboratory Technicians and Sales Executives recorded the highest attrition counts.
* Most attrition occurred during the early years at the company.
* Male employees accounted for a larger share of attrition.
